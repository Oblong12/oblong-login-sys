# oblong-login-sys

A clean, labeled PHP & MySQL login system starter kit.

## Features
- Login / Sign Up with user roles (`user`, `staff`, `manager`).
- Session-based authentication.
- Simple, modern CSS.
- Fully labeled code for easy customization.

## Setup
1. Create a database named `login`.
2. Run the SQL in `setup_db.sql`.
3. Configure `db_conn.php`.

Enjoy!
